# 说明
Railgun为一款GUI界面的渗透工具，将部分人工经验转换为自动化，集成了渗透过程中常用到的一些功能，目前集成了端口扫描、端口爆破、web指纹扫描、漏洞扫描、漏洞利用以及编码转换功能，后续会持续更新。

详细功能可查看pdf文档。

插件及插件化开发参考https://github.com/lz520520/railgunlib
